# Student Management System (QLHS)

## Giới thiệu
Dự án xây dựng hệ thống quản lý học sinh, hỗ trợ quản lý thông tin học sinh, lớp học và dữ liệu liên quan.  
Hệ thống giúp tổ chức và xử lý dữ liệu một cách hiệu quả trong môi trường giáo dục.

---

## Vai trò
- Tham gia phân tích yêu cầu hệ thống  
- Xây dựng Use Case (UML)  
- Thiết kế cơ sở dữ liệu (SQL)  
- Hỗ trợ viết tài liệu mô tả chức năng (SRS)  

---

## Công nghệ sử dụng
- SQL Server / MySQL  
- UML (Use Case, Activity Diagram)  

---

## Chức năng chính
- Quản lý thông tin học sinh  
- Quản lý lớp học  
- Lưu trữ và truy vấn dữ liệu  
- Hỗ trợ theo dõi thông tin học tập  

---

## Nội dung trong repository
- Database: thiết kế và truy vấn SQL  
- Tài liệu: mô tả chức năng hệ thống (SRS)  
- UML: sơ đồ Use Case  

---

## Ghi chú
Đây là dự án học tập theo nhóm nhằm rèn luyện kỹ năng phân tích hệ thống và làm việc với dữ liệu.
